# BMP APK Analyzer

Find Akamai BMP version used in an APK file.

## Prerequisites

Make sure you have [apktool](https://apktool.org/) and [python3](https://www.python.org/) installed.

## Usage

```shell
$ ./bapka <path-to-apk.apk>
```
